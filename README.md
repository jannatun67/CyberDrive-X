## 🌟 Overview

**VELOCITY** is a fully interactive 3D car racing showcase website built with modern web technologies. It features real-time 3D rendering, game-like HUD dashboard, keyboard controls, and a stunning dark/neon gaming aesthetic — all running directly in the browser.

---

## 🎮 Features

| Section | Description |
|---|---|
| 🏠 **Hero** | Full-screen video background + 3D car + CTA |
| ⚡ **Features** | 6 animated glassmorphism cards |
| 🔄 **3D Showcase** | Interactive 360° car viewer |
| 📊 **HUD Dashboard** | Real-time speedometer, gear, nitro gauges |
| 🖼️ **Gallery** | Filterable track gallery |
| 📈 **Specs** | Animated stat bars + vehicle profile |
| 📍 **Footer** | CTA banner + links + credits |

### 🧊 3D Highlights
- Custom car model with floating effect & 360° rotation
- Dynamic neon lighting & particle system
- Speed-reactive wheel spinning & engine vibration
- Reflective ground with grid overlay

### 🎨 UI Highlights
- Glassmorphism design system
- 5 switchable neon color themes
- Cinematic loading screen
- Smooth spring-based scrolling
- Fully responsive (mobile + desktop)
